# صائغ العقود السوري – نظام قرارات قانونية أوفلاين

## الإصدار 1.0.0

تطبيق قانوني شامل للمحامين والمكاتب العقارية في سوريا، يعمل أوفلاين بالكامل.

## المميزات الرئيسية

- **٢٢ شاشة قانونية متخصصة** لإنشاء ومراجعة جميع أنواع العقود
- **١٢ قالب عقد جاهز** للاستخدام المباشر
- **حاسبة مواريث كاملة** تعتمد على ٢٤٠٠ سهم مع عرض الكسور بالأرقام العربية
- **٢٦ تحذير قانوني** (لا تمنع توليد العقد، تظهر داخل التطبيق فقط)
- **٧٧ سيناريو قانوني** مغطى تلقائياً
- **تصدير متعدد الصيغ**: PDF، TXT، PNG
- **قالب فارغ** مخصص للكتابة بالقلم الأزرق
- **٥ خطوط عربية** للاختيار من بينها
- **٣ أحجام ورقية**: A4، A5، Letter
- **تكبير/تصغير الخط** في شاشة المعاينة
- **نظام تتبع وتدقيق** كامل مع بصمة SHA256
- **يعمل أوفلاين ١٠٠٪** بدون أي اتصال إنترنت

## المواصفات التقنية

- Flutter SDK 3.13.x
- Dart >= 3.1.0
- Android 5.0+ (API 21)
- معمارية armeabi-v7a (32-bit) – Samsung A11
- تخزين SQLite محلي
