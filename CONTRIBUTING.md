# المساهمة

نرحب بالمساهمات لتحسين صائغ العقود السوري.

## الخطوات
1. Fork المشروع
2. أنشئ فرعاً (`git checkout -b feature/AmazingFeature`)
3. Commit التغييرات (`git commit -m 'Add some AmazingFeature'`)
4. Push للفرع (`git push origin feature/AmazingFeature`)
5. افتح Pull Request

## الترميز
- اتبع معايير Dart Style.
- اكتب تعليقات بالعربية.
- تأكد من دعم RTL في كل واجهة.
