# ملفات الخطوط العربية المطلوبة

يطلب التطبيق وجود الملفات التالية في هذا المجلد لعرض النصوص العربية بشكل صحيح:

1. TraditionalArabic.ttf (الخط الأساسي)
2. Amiri-Regular.ttf + Amiri-Bold.ttf
3. Cairo-Regular.ttf + Cairo-Bold.ttf
4. NotoNaskhArabic-Regular.ttf + NotoNaskhArabic-Bold.ttf
5. Lateef-Regular.ttf

يمكن تنزيل هذه الخطوط من المواقع الرسمية لكل خط، أو من Google Fonts.

إذا لم تتوفر بعض هذه الملفات، سيعود التطبيق إلى خط TraditionalArabic كخط احتياطي.
